Linux gdb 常用命令

(GNU Debugger)

参数 参数含义 示例
-g 激活调试模式
(告诉编译器这个程序今后可能需要调试)		 gcc -g 1.c -o 11
gdb 启动gdb gdb 11
gdb [executable name]
help 查看帮助 (gdb) help [command]
start 启动程序 (gdb) start
run 运行程序 (gdb) run
break 设置断点 (gdb) break 3
(gdb) break [source code line number]
list 显示和断点相邻的 10 行代码 (gdb) list
next 下一步
（若遇到函数的调用语句，next 会按顺序执行下一条语句）		 (gdb) next
step 下一步
（若遇到函数的调用语句，step 会转而进入函数内部执行接下来的语句）		 (gdb) step
